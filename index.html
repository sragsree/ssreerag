Geekspad - An Android App for Tech Meetups, Recruiter Assistnace and Techie Connect

An android based application for tech geeks to connect, organise meetups and other social events. What makes it really cool is that people can rate other people on their technical skills with reference to their interactions with the person at some tech event or some other means. SDSU Hometown Connect

Hometown is a basically a chat application that helps students to locate people in their hometown who goes to SDSU. This app communicates with SDSU bismarck Server to retrieve students details and firebase for saving chat history.The app has 3 view options List view, Map View, Chat View (WhatsApp like View). The application is extremely fast and responsive as data is cached in SQLite database. Only authorized users can start a conversation. Authentication is implemented using firebase.List View displays the list of users in with their nickname, city, state and country and year of study at SDSU, clicking on a particular user in the list will start a chat with him. Map View plots all users in google map with markers, clicking on a particular marker title will start a conversation with him/her. You can also apply filters like country, state and year to locate a particular user.

Draw & Play - A Stress Reliever App

The days of simple swiping through the menu of your smartphone is over. This magical android application helps you to relieve your stress by allowing you to draw circles of different sizes on the screen and set them in motion. The application basically has 3 modes Insert, delete and Move. In Insert mode, you can draw circles of different sizes on the screen. Delete mode allows you to delete any circle on the screen when you touch them. Move mode sets these circles in motion when you swipe over them with a certain velocity. They collide at the edge of the screen or collide each other to bounce back. The bounce angle is determined by applying laws of physics and each circle loses a certain velocity when they collide each other or with the edge of the screen. How does it help to relieve stress? Rather than swiping through the menu you can draw multiple circles and make them move in different directions. Studies have proven that this kind of apps in mobile are equivalent to stress relief balls.

Multi-Level Imperceptible encryption algorithm
A comparative study of encryption algorithms by implementing encoders and decoders for different algorithms like LSB coding, Parity coding, Phase coding, Echo coding and Spread Spectrum coding to hide images behind audio signals. Their performances were evaluated in terms of Signal to Noise Ratios(SNR). Developed a multi-level imperceptible coding algorithm that is better than the above algorithms in terms of SNR using the same set of data and audio signal. This new algorithm was proved to enhance the performance of computers that works in a network that deals with a lot of encrypted data.

Low Cost Robotic Palm with Arduino
A low-cost implementation of a robotic palm that replicated all the gestures of a human hand. To keep the cost down, the mechanical palm was made with recyclable plastics and homemade etching was followed to design the circuits. Arduino, an embedded unit was used as the controller. Flex Sensors provided input for the Arduino and servo motors were fed with digital outputs from Arduino to mimic the gestures. The Arduino that detected the human gestures and the other Arduino that drove the servo motors communicated wirelessly using ZigBee protocol. An attempt to interface two Arduinos through the Internet by using ZigBee gateway controller modules and TCP/IP protocols was also done as part of this project.

Biomorphic Palm with Arduino
A mechanical palm that replicated all the gestures of a human hand using Arduino(Embedded System). At the transmitter, inputs were read from flex sensors and serialized it before transmitted it with ZigBee module. At the receiver, data was deserialized and passed to servo motors which rotated accordingly to imitate the gestures.
